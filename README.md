# home-mk-resource

## 项目简介

`home-mk-resource` 是一个用于归档和发布独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态资源与页面文件的集中存放与分发用途。

## 目录说明

```
home-mk-resource/
├── pages/          # 存放独立的 HTML 页面文件
├── assets/         # 页面引用的样式、脚本、图片等资源
├── archive/        # 历史版本或不再活跃的页面归档
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，命名清晰，便于定位。
- `assets/`：公共资源文件，按类型或用途组织，供页面引用。
- `archive/`：存放已下线或未维护的页面备份，保留历史记录。

## 页面归档说明

- 所有页面均为独立 HTML 文件，资源引用使用相对路径或数据内嵌方式。
- 页面不依赖特定后端服务，可直接在浏览器中打开或部署于任意静态托管平台。
- 归档内容可能包含示例、工具页面、信息展示页等，不保证长期可用或更新。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件查看效果。
3. 也可将整个仓库部署至静态服务器或 GitHub Pages 使用。
